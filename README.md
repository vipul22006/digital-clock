# 🕒 Analog Clock

A responsive Analog Clock built using HTML, CSS, and JavaScript. The clock displays the current time with smoothly moving hour, minute, and second hands.

## ✨ Features

- Real-time clock
- Smooth hand animations
- Responsive design
- Clean and modern UI
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
